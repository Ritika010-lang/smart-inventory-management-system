# Smart Inventory Management System

## Project Overview

Smart Inventory Management System is a Flask-based web application developed to manage products and inventory efficiently. It allows users to add, update, delete, search, and filter products through an interactive dashboard.

---

## Features

- Login Authentication
- Dashboard
- Product Management (CRUD)
- Add Product
- Edit Product
- Delete Product
- Product Search
- Category Filter
- Status Filter
- Inventory Management
- CSV Export
- Docker Support
- GitHub Actions CI/CD Configuration

## Installation

Clone the repository

```bash
git clone <repository-url>
```

Move into the project

```bash
cd smart-inventory-management-system
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open your browser

```
http://127.0.0.1:5000
```
